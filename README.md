# Features

- [x] Listar pontos do mapa
- [x] Listar dados de cada ponto de acordo com a data local
- [x] Listar dados de usuário em menu lateral 
- [x] Trocar texturas do mapa

OBS: Todos os dados foram fornecidos por uma API externa
